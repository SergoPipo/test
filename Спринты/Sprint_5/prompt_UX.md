### 1.1 Список активных сессий

```
┌─────────────────────────────────────────────────────────────────────┐

├─────────────────────────────────────────────────────────────────────┤
│                                                                     │

│  │ SBER               │  │ GAZP               │  │ LKOH         │  │
│  │ SMA Crossover      │  │ RSI Bounce         │  │ MACD Div     │  │
│  │ P&L: +12 450 ₽     │  │ P&L: -3 200 ₽     │  │ P&L: 0 ₽    │  │
```

- **PAPER** — жёлтая плашка (`yellow.6`), **REAL** — красная плашка (`red.6`)

- Sparkline — мини-график equity за последние 24ч (Lightweight Charts или SVG)
- Кнопки: Пауза, Стоп, График (переход к графику инструмента)
- Пустое состояние: "Нет активных сессий. Запустите первую →"

## 3. Панель «Избранные инструменты» (★)

Расположение: **справа от графика** на странице ChartPage.

```
┌───────────────────────────────────┬────────────────────┐
│                                   │ ★ Избранные        │
│         Свечной график            │                    │
│         (Lightweight Charts)      │ SBER               │
│                                   │ 3 бэкт · 2 стр    │
│                                   │ P&L: +5,2%         │
│                                   │ ────────────────── │
│                                   │ GAZP               │
│                                   │ 1 бэкт · 1 стр    │
│                                   │ P&L: -1,8%         │
│                                   │ ────────────────── │
│                                   │ LKOH               │
│                                   │ 5 бэкт · 3 стр    │
│                                   │ P&L: +12,1%        │
│                                   │                    │
│                                   │ [+ Добавить]       │
└───────────────────────────────────┴────────────────────┘
```

Требования:
- Ширина панели: ~200px, сворачиваемая (toggle кнопкой ★ в toolbar графика)
- Каждый элемент: тикер (жирный), кол-во бэктестов / стратегий, средний P&L
- **Клик по тикеру** → переключение графика на этот инструмент
- **Кнопка «+ Добавить»** → поиск инструмента (тот же search что в LaunchSessionModal)
- **Удаление** — иконка ✕ при hover
- P&L зелёный/красный по знаку
- Данные: средний P&L по всем бэктестам этого тикера

## 4. Улучшения графика (Real-time)

На существующем `CandlestickChart.tsx`:
- **Real-time свечи** через WebSocket канал `market:{ticker}:{tf}` — свечи обновляются в реальном времени
- **Маркеры сделок**: зелёные ▲ (покупка) и красные ▼ (продажа) на графике при наличии активной сессии
- **Индикатор подключения**: мигающий зелёный/красный индикатор в углу графика (WS connected/disconnected)

## 5. Состояния

Для каждого экрана описать 4 состояния:
