## Дизайн-система

- **UI-библиотека:** Mantine 7.8+
- **Тема:** Dark theme (обязательно)
- **Фон приложения:** `#1A1B1E` (Mantine dark)
- **Карточки/панели:** `#25262B`
- **Иконки:** Tabler Icons
- **Шрифт:** системный (Mantine default)
- **Формат чисел:** `1 234 567,89 ₽` (ru-RU)
- **Формат дат:** `ДД.ММ.ГГГГ`

### Цветовая система

| Назначение | Цвет | Mantine token |
|-----------|------|---------------|
| Paper Trading | `#FAB005` | yellow |

## Экран 1: Общий Layout (AppShell)

### Layout

```
┌──────────────────────────────────────────────────────────────┐
│ Header (56px)                                                │
├────────────┬─────────────────────────────────────────────────┤
│ Sidebar    │ Content Area                                    │
│ 240px      │                                                 │
│ (сворач.   │                                                 │
│  до 60px)  │                                                 │
├────────────┴─────────────────────────────────────────────────┤
│ Footer / Status Bar (32px)                                   │
└──────────────────────────────────────────────────────────────┘
```

### Sidebar (240px → 60px)

- **Сворачиваемый:** кнопка-гамбургер в header или chevron внизу sidebar
- **Развёрнутый (240px):** иконка + текст
- **Свёрнутый (60px):** только иконка, tooltip с названием при наведении

**Пункты меню:**
| Иконка | Название | Роут |
|--------|---------|------|
| IconChartBar | Стратегии | `/` (главный дашборд) |
| IconChartLine | Бэктесты | `/backtests` |
| IconTrendingUp | Торговля | `/trading` |
| IconWallet | Счёт | `/account` |
| IconBell | Уведомления | `/notifications` |
| IconSettings | Настройки | `/settings` |

**Mantine-компоненты:** `AppShell.Navbar`, `NavLink`, `Tooltip`

### Header (56px)

- **Слева:** логотип «MOEX Terminal» (текст или SVG)
- **Центр:** поиск инструмента (`Autocomplete` или `Spotlight`)
- **Справа:** колокольчик уведомлений (`ActionIcon` + `Indicator` с count), аватар/профиль (`Menu`)

**Mantine-компоненты:** `AppShell.Header`, `Group`, `Autocomplete`, `ActionIcon`, `Indicator`, `Menu`, `Avatar`

### Footer / Status Bar (32px)

- **Слева:** статус подключения к MOEX (`🟢 Online` / `🔴 Offline`)
- **Центр:** текущий режим (`Paper Trading` жёлтым / `Real Trading` красным)
- **Справа:** время до закрытия торгов (`До закрытия: 2ч 15м`)

**Mantine-компоненты:** `AppShell.Footer`, `Group`, `Badge`, `Text`

### Состояния

- **Loading:** `Skeleton` по всему Content Area
- **Error:** `Alert` с иконкой `IconAlertTriangle`, цвет red
- **Empty:** Content Area пуст, sidebar и header на месте


### Колонки (5 штук)

| Колонка | Тип | Описание |
|---------|-----|----------|
| **Название / Инструмент** | String | Родитель: название стратегии. Дочерняя: [иконка] ТИКЕР  Полное название |
| **Статус** | Badge | Только у дочерних строк. Цвет по типу (см. цветовую систему) |
| **Бэктест** | Text | Дата + PF (Profit Factor) + Max DD (Max Drawdown). Только у дочерних |
| **Позиция** | Text | Объём позиции + абсолютный P&L в скобках. Цвет по знаку |
| **P&L** | Text | Процент прибыли/убытка. Цвет: green (+), red (-), gray (—) |

### Раскрываемые строки

- **Родительская строка:** название стратегии, суммарные Позиция и P&L
- **Дочерние строки:** по одной на каждый инструмент (тикер)
- **▼ / ▶:** иконка раскрытия слева от названия
- Даже если один тикер — всё равно раскрываемая схема (единообразие)

### Иконки тикеров

- Логотип компании 20×20px, загружается по тикеру
- Fallback: первая буква тикера в круге (`Avatar` с `size="xs"`)

### Цветовая кодировка строк

- P&L > 0 → текст Позиции и P&L: `#40C057` (green)
- P&L < 0 → текст Позиции и P&L: `#FA5252` (red)
- Нет данных → `#868E96` (gray)
- Родительская строка — цвет по суммарному P&L

### Кликабельные элементы

| Элемент | Куда ведёт | Роут | Спринт |
|---------|-----------|------|--------|
| Название стратегии | Страница стратегии (редактор, версии) | `/strategy/:id` | S2 |
| Тикер / название инструмента | Страница инструмента (график, стакан) | `/instrument/:ticker` | S3 |
| Данные бэктеста (PF, DD, дата) | Результаты бэктеста (equity curve, сделки) | `/backtest/:id` | S2 |
| Позиция (сумма) | Детали торговой сессии (ордера, журнал) | `/session/:id` | S3 |
| Статус (badge) | Не кликабелен | — | — |
| P&L | Не кликабелен | — | — |

Стиль ссылок: `cursor: pointer`, underline on hover, цвет `blue.4`.

### Контекстное меню строки (правый клик)

| Действие | Иконка |
|----------|--------|
| Редактировать | IconEdit |
| Запустить бэктест | IconChartLine |
| Запустить Paper | IconPlayerPlay (yellow) |
| Запустить Real | IconPlayerPlay (red) |
| Архивировать | IconArchive |
| Удалить | IconTrash (red) |

### Действия

- **Кнопка «+ Новая стратегия»**: `Button` variant="filled", справа вверху. Открывает мастер создания (S2).
- **Фильтр по статусу:** `Select` / `MultiSelect` над таблицей
- **Сортировка:** клик по заголовку колонки (иконка стрелки)
- **Без пагинации** — виртуализация при >100 стратегий (TanStack Table + виртуализация)

### Mantine-компоненты

`Table`, `Badge`, `Avatar`, `Button`, `Select`, `Menu`, `ActionIcon`, `Text`, `Group`, `Skeleton`

### Состояния

- **Loading:** `Skeleton` rows (5–7 строк-заглушек)
- **Empty:** иконка `IconChartBar` + текст «Нет стратегий» + кнопка «Создать первую стратегию»
- **Error:** `Alert` с текстом ошибки

### 3в. Блокировка (после 5 неудачных попыток)

- Красный `Alert` под заголовком: «Аккаунт заблокирован. Повторите через ММ:СС»
- Все поля и кнопка — `disabled`
- Таймер обратного отсчёта обновляется каждую секунду
- Цвет Alert border: `#FA5252` (red)

### Общие параметры экрана авторизации

- **Расположение:** по центру экрана (горизонтально и вертикально)
- **Карточка:** `Paper` с `shadow="md"`, `radius="md"`, `p="xl"`, ширина 400px
- **Логотип:** текст «MOEX Terminal» + подзаголовок «Торговый терминал MOEX»
- **Поля:** `TextInput` (username), `PasswordInput` (password) с кнопкой показа
- **Кнопка:** `Button` fullWidth, `size="md"`
- **Валидация:** inline ошибки под полями (Mantine built-in)

### Mantine-компоненты

`Paper`, `TextInput`, `PasswordInput`, `Button`, `Alert`, `Title`, `Text`, `Center`, `Stack`

### Состояния

- **Loading:** кнопка с `loading={true}` (spinner)
- **Error (неверный пароль):** `Alert` цвет red: «Неверный логин или пароль»
- **Error (блокировка):** `Alert` цвет red: «Аккаунт заблокирован. Повторите через ММ:СС»
- **Success (first-run):** редирект на дашборд
