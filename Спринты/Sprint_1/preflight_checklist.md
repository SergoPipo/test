## Процедура выполнения preflight

1. Оркестратор (Claude) **автоматически** выполняет проверки 1-4
2. Результат выводится в формате:

```
═══════════════════════════════════════════
  PREFLIGHT CHECK — Sprint 1
═══════════════════════════════════════════

  [✓] Python 3.11.9
  [✓] Node 20.11.0
  [✓] pnpm 9.1.0
  [✓] Git 2.44.0
  [✓] Git repo OK (develop branch, clean)
  [✓] CLAUDE.md exists
  [✓] Project structure OK
  [✓] All 8 prompt files present
  [✓] No previous sprint dependencies

  RESULT: ALL CHECKS PASSED — Sprint 1 ready to launch

═══════════════════════════════════════════
```

3. Если есть проблемы:

```
  [✗] pnpm NOT FOUND

  BLOCKERS:
  1. Установите pnpm: npm install -g pnpm

  RESULT: BLOCKED — resolve 1 issue(s) before starting
```

4. Заказчик видит результат и даёт команду на запуск (или устраняет блокеры).
