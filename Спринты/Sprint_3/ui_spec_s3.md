### 1.5 Кастомный Toolbox (левая панель)

**Mantine-компоненты:**
- `Box` w=240, bg="#25262B", overflow-y=auto, h=100%
- `ActionIcon` — кнопка сворачивания (IconChevronLeft / IconChevronRight)
- `Accordion` — категории блоков (chevronPosition="left")
- `Card` — каждый блок (p="xs", radius="sm", withBorder)
- `Group` — иконка + название в заголовке карточки
- `Text` size="xs" c="dimmed" — описание блока
- `Text` size="xs" — параметры по умолчанию

**Категории и цвета:**

| Категория | Цвет | Иконка | Блоков |
|-----------|------|--------|--------|
| Индикаторы | `#339AF0` (blue) | IconChartLine | 8 |
| Условия | `#FD7E14` (orange) | IconArrowsShuffle | 3 |
| Логика | `#845EF7` (violet) | IconLogicAnd | 3 |
| Сигналы | `#40C057` (green) | IconArrowUpRight | 2 |
| Фильтры | `#FAB005` (yellow) | IconClock | 1 |
| Управление | `#E64980` (pink) | IconCoin | 3 |

**Карточка блока (максимальная детализация):**

```
┌──────────────────────┐
│ 🔵 SMA               │  ← цветной круг + название
│ Скользящая средняя    │  ← описание (dimmed, xs)
│ период: 20, close     │  ← параметры по умолчанию (xs)
└──────────────────────┘
```

**Взаимодействие:**
- **Drag & drop** — перетаскивание карточки на Blockly workspace → создаёт блок
- **Двойной клик** — блок добавляется в центр workspace
- **Hover** — карточка подсвечивается (bg: `#2C2E33`)
- **Dragging** — курсор `grabbing`, ghost-preview

**Mantine-компоненты:**
- `Box` — контейнер, bg="#25262B"

- **Успех**: код + зелёный Alert
- **С предупреждениями**: код + жёлтый Alert
- **Ошибка**: красный Alert с описанием

## Экран 2: Mode B — Fullscreen Modal

```
┌─────────────────────────────────────────────────────────────────────┐
│ Импорт стратегии из шаблона                                   [✕]  │
├─────────────────────────────────┬───────────────────────────────────┤
│                                 │ Предпросмотр                      │
│ [📋 Скопировать пустой шаблон]  │                                   │
│                                 │ (пока пусто — вставьте шаблон     │
│ Вставьте заполненный шаблон:    │  и нажмите «Проверить»)           │
│ ┌─────────────────────────────┐ │                                   │
│ │                             │ │                                   │
│ │ 1. НАЗВАНИЕ                 │ │                                   │
│ │ Стратегия SMA Crossover     │ │                                   │
│ │                             │ │                                   │
│ │ 2. ИНСТРУМЕНТЫ              │ │ После нажатия «Проверить»:        │
│ │ SBER, GAZP                  │ │                                   │
│ │                             │ │ Распознанные блоки:               │
│ │ 3. ТАЙМФРЕЙМ                │ │ ✅ SMA (period=20, close)         │
│ │ Дневной (D)                 │ │ ✅ EMA (period=50, close)         │
│ │                             │ │ ✅ Crossover вверх (SMA → EMA)    │
│ │ 4. ИНДИКАТОРЫ               │ │ ✅ Сигнал: Вход Long             │
│ │ - SMA(period=20)            │ │ ✅ Crossover вниз (SMA → EMA)    │
│ │ - EMA(period=50)            │ │ ✅ Сигнал: Выход                 │
│ │                             │ │ ✅ Стоп-лосс: 2%                 │
│ │ 5. УСЛОВИЯ ВХОДА            │ │                                   │
│ │ - SMA(20) пересекает        │ │ Предупреждения:                   │
│ │   EMA(50) снизу вверх       │ │ ⚠ Секция «ФИЛЬТРЫ» не распознана │
│ │   → Long                    │ │                                   │
│ │ ...                         │ │                                   │
│ └─────────────────────────────┘ │                                   │
│                                 │                                   │
│        [Проверить]  [Отмена]  [Применить]                           │
└─────────────────────────────────┬───────────────────────────────────┘
```

**Mantine-компоненты:**
- `Modal` fullScreen, title="Импорт стратегии из шаблона"
- `Grid` cols={2} — левая и правая панели
- `Button` variant="outline" — «Скопировать пустой шаблон» (IconClipboard)
- `Textarea` — вставка шаблона (autosize, minRows=15)
- `Button` variant="light" — «Проверить» (IconEye) — вызывает API parse-template
- `Stack` — правая панель с результатами
- `List` — распознанные блоки с иконками ✅
- `Alert` color="yellow" — предупреждения парсера
- `Group` position="right" — кнопки «Отмена» + «Применить»
- `Button` variant="filled" — «Применить» (disabled пока не проверен)

**Поведение:**
1. Пользователь нажимает «Скопировать пустой шаблон» → шаблон в буфер, notification
2. Заполняет шаблон во внешнем AI или вручную
3. Вставляет в Textarea
4. Нажимает «Проверить» → API `/parse-template` → предпросмотр справа
5. Если OK → нажимает «Применить» → блоки загружаются в Blockly workspace
6. «Отмена» → закрывает modal без изменений

**Mantine-компоненты:**
- `Title` order={2} — «Стратегии»

- `Table` с `@tanstack/react-table` (как в S1)

- **Редактировать** → navigate `/strategies/:id`

- **Удалить** → Modal подтверждения → DELETE

**Состояния:**
- **Loading**: Skeleton rows (3 строки)
- **Empty**: «Нет стратегий. Создайте первую!» + кнопка «+ Новая стратегия»
- **Error**: Alert red «Не удалось загрузить стратегии»

## Общие компоненты

### Resizable Split Panel

Для split view между Blockly и панелью кода:
- Библиотека: `react-resizable-panels` (или Mantine `ResizableBox`)
- Минимальная ширина Blockly: 400px
- Минимальная ширина панели кода: 250px
- Начальная пропорция: 65% / 35%
- Разделитель: 4px, bg="#373A40", cursor="col-resize", hover bg="#4A4D55"

### Навигация

- Sidebar: пункт «Стратегии» (IconBulb) → `/` (DashboardPage)
- DashboardPage: таблица стратегий + кнопка «+ Новая стратегия»
- StrategyEditPage: `/strategies/new` и `/strategies/:id`
- Кнопка «← Назад к списку» → navigate(-1) или `/`
