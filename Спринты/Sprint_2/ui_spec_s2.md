### Селектор таймфреймов

**Подход:** гибридный — кнопки-избранные + выпадающий список.

- **3 кнопки-избранные (★):** пользователь выбирает 3 "favorite" таймфрейма, они отображаются как отдельные кнопки для быстрого доступа
- **Выпадающий список "Все таймфреймы":** содержит все доступные периоды — 1м, 5м, 15м, 1ч, 4ч, Д, Н, М
- **Избранные по умолчанию:** Д, 1ч, 5м
- **Настройка избранных:** клик правой кнопкой или долгий клик на таймфрейме в выпадающем списке → toggle "★ Избранное"

**Mantine-компоненты:**
- `Button.Group` — для 3 кнопок-избранных
- `Button` — каждая кнопка, `variant="filled"` для активной, `variant="subtle"` для остальных
- `Menu` — выпадающий список всех таймфреймов
- `ActionIcon` + `IconStar` — иконка избранного в каждом пункте Menu

**Хранение:** избранные таймфреймы в `localStorage` (Zustand persist)

### Панель Volume

- **По умолчанию:** 20% высоты графика
- **Регулируемая:** пользователь может перетаскивать разделитель между графиком и volume
- **Реализация:** CSS `resize` или кастомный drag-handler на разделителе
- **Минимум:** 10%, максимум: 40%
- **Хранение:** последний размер в `localStorage`

### Сводка OHLCV (crosshair)

- **Расположение:** над графиком, под селектором таймфреймов
- **Содержимое:** `O: 286,50  H: 289,00  L: 285,20  C: 287,50  V: 42 000`
- **Видимость:** всегда показана (кнопка скрытия убрана по решению заказчика)
- **Обновление:** при движении crosshair — данные текущей свечи под курсором; без курсора — данные последней свечи

**Mantine-компоненты:**
- `Text` size="xs" c="dimmed" — для значений OHLCV

### Информация об инструменте

- **Расположение:** верхняя панель
- **Содержимое:** `SBER · Сбербанк ао   287,50 ₽  +1,23 (+0,43%)`
- **Цвет изменения:** зелёный (`#40C057`) для роста, красный (`#FA5252`) для падения
- **Формат:** цена `1 234,56 ₽`, изменение `+1,23 (+0,43%)`

**Mantine-компоненты:**
- `Group` — горизонтальная группа
- `Text` fw={700} — тикер
- `Text` c="dimmed" — название
- `Text` — цена
- `Text` c="green" / c="red" — изменение

### Переход к графику

- Клик по тикеру в таблице стратегий (DashboardPage) → `navigate(`/chart/${ticker}`)`
- Тикер в таблице: `cursor: pointer`, `text-decoration: underline` при hover

### Состояния

- **Loading:** `Skeleton` на всю область графика, `Loader` по центру
- **Empty (нет тикера):** `Center` + `IconChartCandle` (64px, dimmed) + "Выберите инструмент для отображения графика" + `Text` c="dimmed" "Используйте поиск в заголовке или выберите из списка стратегий"
- **Error:** `Alert` color="red" с сообщением "Не удалось загрузить данные для {ticker}" + кнопка "Повторить"
- **Filled:** полноценный свечной график

## 2. Настройки брокера (BrokerSettingsPage)

**Роут:** `/settings` (вкладка "Брокер")
**Sidebar:** существующий пункт "Настройки" (`IconSettings`, path: `/settings`)

### Layout

```
┌─────────────────────────────────────────────────────┐
│ Настройки                                           │
├──────────┬───────────┬──────────────┐               │
│ ● Брокер │  Профиль  │ Уведомления  │               │
├──────────┴───────────┴──────────────┴───────────────┤
│                                                     │
│  Подключённые аккаунты         [+ Добавить брокера] │
│ ┌───────────────────────────────────────────────────┐│
│ │ Мой брокерский │ T-Invest │ Брокер │ 1 234 567 ₽ ││
│ │                │          │        │ Активен 🟢   ││ ✏️ 🗑️
│ ├───────────────────────────────────────────────────┤│
│ │ ИИС            │ T-Invest │ ИИС    │   500 000 ₽ ││
```

### Вкладки настроек

**Mantine-компоненты:**
- `Tabs` — контейнер вкладок
- `Tabs.Tab` — "Брокер" (активная), "Профиль" (заглушка S1), "Уведомления" (заглушка S6)

Вкладки "Профиль" и "Уведомления" — заглушки с текстом "Будет доступно в следующих обновлениях".

### Кнопка "Добавить брокера"

- `Button` variant="filled" leftSection={`IconPlus`}
- Открывает модальное окно с формой

### Модальное окно: Добавление брокера

```

│ Добавить брокера                    ✕  │
├─────────────────────────────────────────┤

```

**Mantine-компоненты:**
- `Modal` size="md" title="Добавить брокера"
- `Select` — тип брокера (data: [{value: "tinvest", label: "T-Invest (Тинькофф)"}])
- `TextInput` — название аккаунта
- `PasswordInput` — API-ключ (с toggle видимости)
- `PasswordInput` — API-секрет
- `Checkbox` — Sandbox-режим
- `Button` variant="outline" — "Обнаружить счета" (loading state при запросе)
- `Checkbox.Group` — список найденных счетов
- `Button` variant="filled" — "Сохранить"
- `Button` variant="subtle" — "Отмена"

**Валидация:**
- broker_type: обязателен
- name: обязателен, 1-100 символов
- api_key: обязателен, минимум 10 символов
- Ошибка валидации: красный текст под полем (Mantine input error)

### Список аккаунтов

**Mantine-компоненты:**
- `Table` — таблица аккаунтов (или `Card` для каждого аккаунта)
- Колонки: Имя, Брокер, Тип (Брокерский/ИИС), Баланс, Статус, Действия
- Баланс: формат `1 234 567,89 ₽`
- Статус: `Badge` size="xs" — Активен (green) / Неактивен (gray)
- Действия: `ActionIcon` × 2 — редактировать (`IconEdit`), удалить (`IconTrash`)

### Sandbox-индикатор

- Если `config.TINVEST_SANDBOX === true`: показывать `Alert` color="yellow" icon={`IconAlertTriangle`}
- Текст: "Sandbox-режим активен — используются тестовые данные. Переключите в настройках для работы с реальным счётом."

### Модальное окно: Редактирование

- `Modal` size="sm" title="Редактировать аккаунт"
- `TextInput` — имя
- `Switch` — Активен/Неактивен
- Кнопки: "Отмена", "Сохранить"

### Модальное окно: Удаление

- `Modal` size="sm" title="Удалить аккаунт"
- `Text`: "Вы уверены, что хотите удалить аккаунт «{name}»? Это действие необратимо."
- `Button` color="red" — "Удалить"
- `Button` variant="subtle" — "Отмена"

### Empty state

- `Center` + `IconWallet` (64px, dimmed)
- `Text` size="lg": "Нет подключённых брокеров"
- `Text` c="dimmed": "Добавьте брокерский аккаунт для загрузки рыночных данных и торговли"
- `Button` variant="filled": "+ Добавить брокера"

### Состояния

- **Loading:** `Skeleton` для таблицы
- **Empty:** описано выше
- **Error:** `Notification` color="red" с текстом ошибки
- **Success:** `Notification` color="green" "Аккаунт добавлен" / "Аккаунт удалён"

## 3. Динамический Footer

### Layout

```
┌────────────────────────────────────────────────────────────────────┐
│ 🟢 MOEX Online │ 🔗 Брокер: подключён │ Paper Trading │ До закрытия: 2ч 15м │ 15:45 МСК │
└────────────────────────────────────────────────────────────────────┘
```

### Элементы (слева направо)

**1. Статус MOEX:**
- `Badge` size="xs" variant="dot"
- Варианты:

| Статус | Dot color | Текст |
|--------|-----------|-------|
| online | green | MOEX Online |
| evening | yellow | MOEX Вечерняя |
| auction | orange | MOEX Аукцион |
| offline | red | MOEX Offline |
| weekend | gray | Выходной |

**2. Статус брокера:**
- `Text` size="xs"
- `IconLink` (connected) / `IconLinkOff` (disconnected) — size 12
- "Брокер: подключён" (green) / "Брокер: нет подключения" (dimmed)
- Если нет аккаунтов: не показывать

**3. Режим торговли:**
- `Text` size="xs"
- "Paper Trading" — color yellow (`#FAB005`)
- "Real Trading" — color red (`#FA5252`) (будет в S5)
- "Нет активных сессий" — c="dimmed"

**4. Время до закрытия:**
- `Text` size="xs" c="dimmed"
- "До закрытия: Xч Yм" — если рынок открыт
- "Торги завершены" — если рынок закрыт
- "Выходной" — если нерабочий день

**5. Время Москвы:**
- `Text` size="xs" c="dimmed"
- Формат: `HH:MM МСК`
- Обновление: каждые 60 секунд

### Обновление данных

- Endpoint: `GET /api/v1/market-data/market-status` (публичный)
- Интервал: каждые 60 секунд (`setInterval`)
- Время Москвы: вычисляется на клиенте (`Intl.DateTimeFormat` с timezone Europe/Moscow)

### Mantine-компоненты

- `Group` justify="space-between" px="md" h="100%" gap="lg"
- `Badge` — статус MOEX
- `Text` size="xs" — остальные элементы
- `Divider` orientation="vertical" — разделители между секциями (опционально)

## 4. Поиск инструмента в Header

### Layout

```
┌──────────────────────────────────────────┐
│ 🔍 Поиск инструмента...                 │
├──────────────────────────────────────────┤
│ Недавние:                                │
│   SBER  Сбербанк ао            акция    │
│   GAZP  Газпром                акция    │
├──────────────────────────────────────────┤
│ (ввод >= 2 символов для поиска)          │
└──────────────────────────────────────────┘

после ввода >= 2 символов:

┌──────────────────────────────────────────┐
│ 🔍 SB                                    │
├──────────────────────────────────────────┤
│ SBER   Сбербанк ао             акция    │
│ SBERP  Сбербанк ап             акция    │
└──────────────────────────────────────────┘
```

### Поведение

- **Фокус без ввода:** показать последние просмотренные инструменты (до 5 шт), заголовок "Недавние:" (dimmed)
- **Ввод < 2 символов:** подсказка "Введите минимум 2 символа"
- **Ввод >= 2 символов:** запрос `GET /market-data/instruments?query={text}` с debounce 300ms
- **Результаты:** тикер (bold) + название + тип (badge: акция/облигация/ETF)
- **Клик по результату:** `navigate(`/chart/${ticker}`)`, добавить в "Недавние"
- **Escape:** закрыть dropdown
- **Пустой результат:** "Ничего не найдено"

### Хранение

- Недавние инструменты: `localStorage` (массив тикеров, макс 5)
- Zustand persist или прямой доступ к `localStorage`

### Mantine-компоненты

- `Autocomplete` — основной компонент (заменяет disabled `TextInput`)
- `IconSearch` size={16} — leftSection
- `Group` + `Text` — каждый пункт выпадающего списка
- `Text` fw={700} — тикер
- `Text` c="dimmed" — название
- `Badge` size="xs" variant="outline" — тип (акция/облигация/ETF)

### Цвета badge по типу

| Тип | Color | Label |
|-----|-------|-------|
| Акция | blue | акция |
| Облигация | orange | облигация |
| ETF | grape | ETF |
| Фьючерс | red | фьючерс |
