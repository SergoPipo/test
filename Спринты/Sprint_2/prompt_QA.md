# Задачи

## 1. Запуск всех тестов

Первым делом запусти все существующие тесты:

```bash
# Backend
cd backend && pip install -e .[dev] && pytest tests/ -v --tb=short

# Frontend
cd frontend && pnpm test
```

Зафиксируй результат: сколько тестов, сколько passed, сколько failed.

Создай файл `Sprint_2/test_cases_s2.md` со следующими тест-кейсами:

|-----|----------|----------|

## 3. Ручное тестирование

Выполни тест-кейсы TC-2.01 — TC-2.12 и зафиксируй результаты:

```
| TC | Статус | Комментарий |
|----|--------|-------------|
| TC-2.01 | ✅/❌ | |
| TC-2.02 | ✅/❌ | |
| ... | | |
```

## 4. Проверка безопасности

- [ ] API-ключи **не логируются** (проверить structlog output)
- [ ] API-ключи **не возвращаются** в API-ответах
- [ ] API-ключи **зашифрованы** в БД (не plaintext)
- [ ] CORS настроен корректно
- [ ] Все endpoints (кроме market-status) требуют JWT

## 5. Проверка финансовых данных

- [ ] Цены в формате `Decimal` (backend), не `float`
- [ ] Числа на frontend: `1 234,56 ₽` (ru-RU)
- [ ] Даты на frontend: `ДД.ММ.ГГГГ`
